# Version

<Intro>
The `version` command allows you to manage the currently installed versions of Jumppad.
</Intro>

## Target specific versions

To allow compatability between the version of Jumppad a user has installed and the version ed to create resources, Jumppad allows you to create resources with versions other than the main installed version.

This capability is either executed automatically when a blueprint author specifies the `version` in the jumppad resource.

```hcl
jumppad {
  version = "^0.1.4"
}
```

## Usage

```shell
jumppad version [command] [flags]
```
