# Status

<Intro>
Show the status of the current stack.
</Intro>

## Usage

```shell
jumppad status
```

## Flags

<Properties>
  <Property name="json">
    Output the status as JSON.
  </Property>
  <Property name="type" type="string" required="false" value="">
    Resource type used to filter the status list.
  </Property>
</Properties>
