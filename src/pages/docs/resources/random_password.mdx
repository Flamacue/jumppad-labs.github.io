# Random Password `random_password`

<Intro>
The `random_password` resource allow the creation of random passwords.
</Intro>

## Properties

<Properties>
  <Property name="length" type="int" required="true" value=""> 
    The length of the string desired. The minimum value for length is 1 and, length must also be >= (`min_upper` + `min_lower` + `min_numeric` + `min_special`).
  </Property>

  <Property name="special" type="bool" value="true"> 
    Include special characters in the result. These are `!@#$%&*()-_=+[]{}<>:?`.
  </Property>

  <Property name="min_special" type="int" value="0"> 
    Minimum number of special characters in the result.
  </Property>

  <Property name="override_special" type="string" value=""> 
    Supply your own list of special characters to use for string generation.
    This overrides the default character list in the special argument.
    The special argument must still be set to true for any overwritten characters to be used in generation.
  </Property>
  
  <Property name="numeric" type="bool" value="true"> 
    Include numeric characters in the result.
  </Property>

  <Property name="min_numeric" type="int" value="0"> 
    Minimum number of numeric characters in the result.
  </Property>

  <Property name="lower" type="bool" value="true"> 
    Include lowercase alphabet characters in the result.
  </Property>

  <Property name="min_lower" type="int" value="0"> 
    Minimum number of lowercase alphabet characters in the result.
  </Property>

  <Property name="upper" type="bool" value="true"> 
    Include uppercase alphabet characters in the result.
  </Property>

  <Property name="min_upper" type="int" value="0"> 
    Minimum number of uppercase alphabet characters in the result.
  </Property>

  <Property name="value" type="string" value="" readonly> 
    The generated random password.
  </Property>
</Properties>

<MetaProperties/>

## Examples

```hcl
resource "random_password" "password" {
    length = 32
}

output "password" {
    value = resource.random_password.password.value
}
```
