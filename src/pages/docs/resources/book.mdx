# Book `book`

<Intro>
Bundles `chapter` resources together so it can be displayed in the `docs` resource.
</Intro>

<Properties>
  <Property name="title" type="string" required="true" value="">
    The title of the book.
  </Property>

  <Property name="chapters" type="[]string" required="true" value="">
    A list containing the IDs of `chapter` resources included in the book.
  </Property>

  <Property name="index" type="string" value="" readonly>
    A JSON index generated from the chapters of the book.
  </Property>
</Properties>
