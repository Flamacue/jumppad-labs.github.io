# Blueprint `blueprint`

Provides metadata about the blueprint configuration.
It is also used by the docs command to automatically generate README files.

We plan to have a searchable repository of blueprints at some point, which will use this metadata.

<Properties>
  <Property name="title" type="string" value="">

  <Property name="author" type="string" value="">
    The author of the blueprint.
  </Property>

  <Property name="slug" type="string" value="">

  </Property>

  <Property name="description" type="string" value="">
    The description of the blueprint.
